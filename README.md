Sunucu çalıştırmak için öncelikle klasörün jar dosyasını çıkarın. Klasörün konsolundayken mvn clean package -DskipTests yazınca jar dosyası çıkar. Bundan sonra docker desktop uygulaması açıkken eatlens klasöründeyken konsola docker-compose up --build yazınca sunucu çalışır localhost:8080 ile erişilebilir.
